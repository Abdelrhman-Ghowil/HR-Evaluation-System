Evaluation Debug Test

Debug Information:

This page is for debugging the evaluation display issue.

Please check the browser console for detailed logs when you navigate to an employee's details page.

The console logs will show:

  • Employee ID being used for the API call
  • Loading and error states
  • Raw API response data
  • Data transformation process
  • Final evaluation list

Expected API Response Structure:

{
  "count": number,
  "next": string | null,
  "previous": string | null,
  "results": [
    {
      "evaluation_id": "string",
      "employee": "string",
      "employee_id": "string",
      "type": "Annual" | "Quarterly" | "Optional",
      "status": "Draft" | "Pending HoD Approval" | ...,
      "score": "string",
      "reviewer": "string",
      "reviewer_id": "string",
      "period": "string",
      "created_at": "string",
      "updated_at": "string"
    }
  ]
}

Troubleshooting Steps:

  1. Open browser developer tools (F12)
  2. Go to Console tab
  3. Navigate to an employee's details page
  4. Look for console logs starting with "=== EVALUATION DATA DEBUG ==="
  5. Check if the API response matches the expected structure above
  6. Verify that the employee_id parameter is being sent correctly
